### Notas de implementação

- **Toggle:** `border-radius` extremamente alto (≈58M px) — usar `borderRadius: BorderRadius.circular(100)` em Flutter para pill completa
- **Borda dashed upload:** Flutter usa `Border` com `BorderStyle.solid`; para dashed usar `CustomPainter` ou `DashedBorder` package
- **Streak calendar:** células com opacidade gradiente — implementar com `Opacity` widget ou `Color.fromRGBO` com alpha variável por dia
- **Skin progress bar tri-segmentada:** usar `Row` com `Expanded` e flex weights [58, 19, 38]
- **Badge cards opacity 50%:** usar `Opacity(opacity: 0.5, child: BadgeCard(...))` para estado bloqueado
- **Borda apenas esquerda (Coach blocks):** `BoxDecoration(border: Border(left: BorderSide(color: Color(0xFFFF6B35), width: 1.735)))`
